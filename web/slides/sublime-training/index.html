What is Sublime Text?

Let's ask Wikipedia:

Sublime Text is a cross-platform text and source code editor, with a Python application programming interface (API). Sublime Text is proprietary software. Its functionality is extendable with plugins. Most of the extending packages have free-software licenses and are community-built and maintained. Sublime Text lacks graphical setting dialogues and is entirely configured by editing text files.

↓

It's that dark multi-colored text editor that's getting attention lately, and used by modern developers around the world.

Why Sublime Text?

Sublime Text is a sophisticated text editor for code, markup and prose.

It is NOT an IDE.

But rather, an ultra-fast, compact, and extensible text editor.

And, it's super COOL.

If you're an IDE-type of developer, or need to work with projects dependent on IDE functionalities (e.g. iOS, Android, etc.), then stick with IDE's. Else, use a text editor for quick edits.

The Features — again from Wikipedia:

  • "Goto Anything," quick navigation to files, symbols, or lines
  • "Command palette" uses adaptive matching for quick keyboard invocation of arbitrary commands
  • Multiple selections: simultaneously make the same interactive changes to multiple selected areas
  • Python-based plugin API
  • Project-specific preferences
  • Extensive customizability via JSON settings files, including project-specific and platform-specific settings
  • Cross platform (Windows, OS X, Linux)
  • Compatible with many language grammars from Textmate.
↓

  • Super quick file search and navigation
  • Instantly jump to symbols, lines, or words.
  • @ to jump to symbols
  • # to search within the file

Multiple Selections

Make ten changes at the same time, not one change ten times.

Multiple selections allow you to interactively change many lines at once, rename variables with ease, and manipulate files faster than ever.

The Package Manager

Sublime Text can be easily extended through the package manager.

This includes things like themes, snippets, to full-blown functionalities not found on the original installation.

↓

Some notable plugins:

  • Emmet (formerly Zen Coding)
  • jQuery (highlight & snippets)
  • LESS and SASS
  • LiveStyle
  • HTMLPrettify
  • CSSComb
  • AngularJS
  • SublimeGit
  • BracketHighlighter
  • Bootstrap 3 Snippets
↓

For more information on the official repository, visit https://sublime.wbond.net/.

Keyboard Shortcuts

Editing

Ctrl+C copy current line (if no selection)
Ctrl+X cut current line (if no selection)
Ctrl+⇧+K delete line
Ctrl+↩ insert line after
Ctrl+⇧+↩ insert line before
Ctrl+⇧+↑ move line (or selection) up
Ctrl+L select line (repeat to select next lines)
Ctrl+D select word (repeat select others occurrences in context for multiple editing)
Ctrl+M jump to closing bracket for current code, repeat to jump to opening bracket
Ctrl+⇧+M select all contents of the current brackets (curly brackets, square brackets, parentheses)
Ctrl+KK delete from cursor to end of line
Ctrl+K+⌫ delete from cursor to start of line
Ctrl+] indent current line(s)
Ctrl+[ un-indent current line(s)
Ctrl+⇧+D duplicate line(s)
Ctrl+J join line below to the end of the current line
Ctrl+ / comment/un-comment current line
Ctrl+⇧+/ block comment current selection
Ctrl+Y redo, or repeat last keyboard shortcut command
Ctrl+⇧+V paste and indent correctly
Ctrl+Space select next auto-complete suggestion
Ctrl+U soft undo (somehow undoes your movements; it jumps to your last change before undoing it when you repeat this command)

Navigation/Goto Anywhere

Ctrl+P quick-open files by name in your project (doesn’t seem to need an actual project set up, it just searches in the directories around the currently-opened file
Ctrl+R goto symbol (functions and classes) in the file. Same as Ctrl+P, then type @
Ctrl+; goto word in current file. Same as Ctrl+P, then type #
Ctrl+G goto line in current file. Same as Ctrl+P, then type :

General

Ctrl+⇧+P command prompt
Ctrl+KB toggle side bar

Find/Replace

Ctrl+F find
Ctrl+H replace
Ctrl+⇧+F find in files

Split window

Alt+⇧+2 split into two columns
Alt+⇧+1 revert to single column
Alt+⇧+5 grid (4 groups)
Ctrl+[1,2,3,4] jump to “group” (pane)
Ctrl+⇧+[1,2,3,4] move file to specified group

Bookmarks

Ctrl+F2 toggle bookmark
F2 next bookmark
⇧+F2 previous bookmark
Ctrl+⇧+F2 clear bookmarks

Text manipulation

Ctrl+KU upper case
Ctrl+KL lower case
