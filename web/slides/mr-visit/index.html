M-R Wordpress Boilerplate

M-R Wordpress Boilerplate

  • A Wordpress boilerplate especially made for MR projects
  • Includes the essential theme and plugins for development
  • Comes with an updated version of the Genesis Theme, along with a simplified version of the Genesis Sample Theme
  • Minimizes the time needed to setup Wordpress for MR clients
  • SASS-ified!
  • Genesis Hooks Debugger to target hooks with ease
